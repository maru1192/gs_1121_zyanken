# ①課題名
サザエさんバトルゲーム

## ②課題内容（どんな作品か）
- サザエさんのバトル形式のゲーム。カツオと花沢さんそれぞれコマンドを4つ持っており戦えるようにしています。

## ③アプリのデプロイURL
- なし

## ④アプリのログイン用IDまたはPassword（ある場合）
- なし

## ⑤工夫した点・こだわった点
- 吹き出し部分をタイプライト形式で表示させた部分
- ボタンを押した後、自分のターンと相手のターンは、ボタンが反応しないようにした点

## ⑥難しかった点・次回トライしたいこと（又は機能）
- ボタンが反応しない時は、ボタンの色を灰色にしたり、相手のターンは相手のターンだと視覚的にわかるような設定をしたいです。
- 元々4キャラクター用意して勝ち抜き戦で考えていたのですが、1バトルまでしか実現できなかったので、勝ち抜き戦の実装もさせたいです。
